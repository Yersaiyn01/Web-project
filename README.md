## Description
Expense Tracker is a web application that allows users to manage their expenses and income.
## Frontend: 
Angular 17, HttpClient for API requests, reactive forms, routing, JWT interceptor.
## Backend: 
Django 4.2, DRF, SQLite database, CORS enabled, JWT authentication.
